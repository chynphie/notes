Fast Fourier Transform

  • Algorithm
Fast Fourier Transform
	function F = FastFT(f, N)
		m = log N
		if m = 0
			F = f
		else
			build g[N]
			build h[N]
		F[even] = (1/2) FastFT(g, N/2)
		F[odd] = (1/2) FastFT(h, N/2)
	end
