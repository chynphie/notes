- **==Theorem 1.7==**: Let T=(U,F) be a tree
	1. For any edge $e\not\in F$, graph $T+e\equiv(U,F\cup \{ e \})$ has exactly one cycle C
	2. For any edge $e'\in C$, graph $T+e-e'\equiv(U,(F\cup \{ e \})-\{ e' \})$ is a tree
- **Fundamental Cycle**: the unique cycle of $T+e$, where e is an edge $e\not\in E(T)$
