
- <u>Identities</u>: An element $e ∈ X$ is an identity for if  $e\boxtimes\:x=x\boxtimes\:e = x$  for all $x ∈ X$
	- Eg: 
		1. $0\in \mathbb{Z}$ is an identity for + since 0+x=x+0=x for all $x\:\in\:\mathbb{Z}$
		2. $1\in \mathbb{Q}$ is an identity for $\cdot$ 
		3. $0\in \mathbb{Q}$ also an identity for $\cdot$
	- **Identities are unique**
		- An element $e\in X$ is an identity for $\boxtimes$ if $e\:\boxtimes\:x=x\:\boxtimes\:e=x\:for\:all\:x\:\in X$ 
> [!lemma] 
> If $e,e'\in X$ are both identities for $\boxtimes$, then $e=e'$

- <u>Inverses</u>: Let $\boxtimes$ be a binary operation on X with an identity element e. 
	- An element y is a **left inverse** for x if $y\:\boxtimes\:x=e$
	- An element y is a **right inverse** for x if $x\:\boxtimes\:y=e$
	- An **inverse ** if $x\:\boxtimes\:y=y\:\boxtimes\:x\:=e$
	- Eg：
		1. -n is an inverse for $n\in \mathbb{Z}$ for +
