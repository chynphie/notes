> [!definition] Binary operation
> a function that combines two elements of a set to produce a third element within the same set

- **Closure:**
    The result of the binary operation on any two elements of the set must also be an element of the set.
- **Binary:**
    It takes two inputs (elements from the set) and produces one output (an element in the set). 

Examples:
- **Addition on the set of integers:** Given any two integers, their sum is also an integer.
- **Multiplication on the set of real numbers:** The product of any two real numbers is also a real number.
- **Set intersection:** The intersection of two sets results in another set (the set of elements common to both).
